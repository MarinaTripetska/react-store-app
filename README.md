# React Store App

Simple store application built with **React** and **TypeScript** as a recruitment task.

## Features
- Product list fetched from Fake Store API
- Client-side routing (Home, Products, 404)
- Product sorting (title, price, default)
- Shopping cart with quantity management
- Persistent cart state using localStorage
- Random product displayed on the Home page
- Loading and error states handling

## Tech Stack
- React
- TypeScript
- React Router
- Fake Store API

## Getting Started

```bash
npm install
npm run dev
